# 测试相关任务

本文件夹存放各类测试开发和执行任务。

## 任务类型

### 单元测试
- Rust模块测试
- Vue组件测试
- Mock数据和服务

### 集成测试
- API接口测试
- 数据库集成测试
- 系统间交互测试

### 端到端测试
- 用户界面自动化测试
- 业务流程测试
- 回归测试套件

### 性能测试
- 负载测试
- 压力测试
- 内存和性能分析

### 安全测试
- 漏洞扫描
- 权限控制测试
- 数据安全验证

## 相关文件

- `tests/` - 测试代码目录
- `web-ui/src/test/` - 前端测试
- `dev/scripts/test_*.ps1` - 测试脚本
