## 🚀 部署指南

### 快速启动
```bash
# 1. 编译项目
cargo build --release

# 2. 运行测试验证
cargo test --workspace

# 3. 启动Docker环境
docker-compose up -d

# 4. 运行网关
./target/release/edge-gateway
```

### 环境要求
- Rust 1.70+
- Docker & Docker Compose
- 8GB+ RAM (推荐)
- Linux/macOS/Windows
